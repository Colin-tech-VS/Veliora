Questions fréquentes

Veliora remplace mon CRM actuel ?

Non pour la gestion globale du parc. Veliora est un CRM mandats orienté prospection : pipeline vendeurs, scoring, mandats, scripts. Gardez Hektor, Apimo ou équivalent pour le reste.

Qu’est-ce que le crawl fait concrètement ?

Il parcourt les URLs de recherche que vous configurez, détecte les annonces, enrichit les fiches (contacts, prix, DVF si vente) et alimente le briefing radar. Veille manuelle, passes automatiques ou import d’une URL ponctuelle (mode analyse).

Le score et la synthèse d’appel, c’est quoi ?

Le Score Mandat™ = règles métier affichées (particulier, baisse, DVF sur vente…). La synthèse d’appel = paragraphes générés à partir de ces signaux (pas un modèle externe type ChatGPT). Chaque critère du score reste visible dans le CRM.

Données partagées entre agences ?

Non. Chaque agence a son espace isolé : sources, prospects, crawls et utilisateurs.
